# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/Jackson-AI-Dev2416/flutter-ai-dashboard.git
```

---

## Install Dependencies

```bash
flutter pub get
```

---

## Run Application

```bash
flutter run
```
